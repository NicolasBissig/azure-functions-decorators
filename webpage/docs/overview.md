# Overview

azure-functions-decorators documentation and guides.

## Getting Started

Install the dependency:

```shell
npm i azure-functions-decorators
```

Enable experimental decorators in your `tsconfig.json`:

```json
{
    // ...
    "compilerOptions": {
        // ...
        "experimentalDecorators": true
    }
}
```
